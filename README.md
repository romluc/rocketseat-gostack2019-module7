# Rocketseat's GoStack2019 - Module07 :rocket:
React e-commerce app, using Redux and Redux-Saga to consume locally hosted RESTful api
### Handles empty cart, out of stock item and submit order

### Features / Dependencies: 
- ReactJS
- Redux
- Redux-saga
- Toastify
- Hooks
- Reactotron 
- Axios
- History
- Immer
- Style-components
- Polished
